# Shokor Academy

A personal website for **Ali Shokor** — Computer Science instructor, AWS Certified trainer, and founder of Shokor Academy.

## Features

- **Hero** — Vanta.js 3D animated network background with social proof stats
- **About** — Instructor bio, experience, and teaching philosophy
- **Services** — AWS certification prep, CS foundations, exam prep & mentoring
- **Courses** — Structured C language curriculum (I1101, I2204, I2206)
- **Resources** — YouTube video tutorials for course content
- **Testimonials** — Student feedback carousel
- **Contact** — Floating-label form with WhatsApp & Email integration
- **Custom cursor** — Purple glow dot, ring, and trailing particles
- **Footer** — Social links (Instagram, LinkedIn, GitHub, YouTube)

## Tech Stack

- HTML5, CSS3, JavaScript (vanilla)
- [Vanta.js](https://www.vantajs.com/) + [Three.js](https://threejs.org/) (hero background)
- [Spline](https://spline.design/) (3D robot in contact section)
- [Google Fonts](https://fonts.google.com/) (Inter)
